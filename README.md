# Poppy Marketing & Consulting - Landing Page

B2B marketing agency landing page showcasing services, portfolio, and expertise for small business marketing.

## Overview
This is a high-converting landing page for Poppy Marketing & Consulting, a digital marketing agency specializing in Google Ads, social media marketing, web design, and brand development for small businesses in the Houston area.

## Features
- ✅ Fully responsive design (desktop, tablet, mobile)
- ✅ Mobile-first approach with 3-tier breakpoints (1024px, 768px, 480px)
- ✅ Professional CSS styling with consistent spacing scale
- ✅ High-converting CTA buttons with visual dominance
- ✅ Service showcase with detailed descriptions
